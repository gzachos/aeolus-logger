Emerson Logger Main Page

About

The "Emerson Logger" project is about a server that will be used to monitor the environmental conditions of the cluster room inside the building of Computer Science and Engineering Department - University of Ioannina. The temperature and humidity values are supplied by the two Emerson cooling units inside the cluster room. Moreover, the project is about monitoring the status of the two Emerson units and alarming the faculty if any abnormal conditions are observed.

More Details

The two emersons provide XML files containing the related data for reporting their status and environmental conditions. Each unit provides: a 24 hour log of the 1) unit temperature, 2) unit humidity, 3) system temperature, 4) system humidity and 5) a current status report. Each measurement is taken every 6 minutes. That equals to 10 measurements per hour, which sums up to a log of 240 measurements. The XML files of Emerson unit #3 are provided 22 minutes after the actual measurement, while the XML files of the #4 unit are provided 18 minutes after the actual measurement.
